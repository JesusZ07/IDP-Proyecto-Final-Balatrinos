Para ejecutar el programa (aplicacion de escritorio y web), abre la solucion en VS Code y realiza lo siguiente:

1.- Haz clic en los tres puntos (...) junto al boton de ejecutar.
2.- Selecciona la opcion Terminal > Run Task.
3.- En el menu, elige run-desktop para la aplicacion de escritorio o run-web para la aplicacion web.

Nota: Es necesario tener en ejecucion los servicios SQL Server (MSSQLSERVER) y SQL Server Agent para que funcione la base de datos.
Tambien necesitas el respaldo Hotel.bak; de lo contrario no podras hacer reservaciones, registrarte ni iniciar sesion, o si gustas crea tu propia base de datos llamada Hotel, pero usa los sql Crear tablas y Llenar tablas

Para ejecutar pruebas con su reporte en un solo comando, solo abran la terminal o una nueva terminal.

Ejecutar todas las pruebas:
dotnet test "HotelProyecto.Tests/HotelProyecto.Tests.csproj" --configuration Release

Ejecutar solo las pruebas unitarias (Framework xunit):
dotnet test "HotelProyecto.Tests/HotelProyecto.Tests.csproj" --configuration Release --filter "FullyQualifiedName~PruebasUnitarias"

Ejecutar solo las pruebas funcionales de UI (Playwright y framework xunit):
dotnet test "HotelProyecto.Tests/HotelProyecto.Tests.csproj" --configuration Release --filter "FullyQualifiedName~PruebasFuncionalesUI"


Si falla la ejecucion de las pruebas UI (Playwright), instalen Chromium con el siguiente comando y una vez instalado vuelvan a ejecutar la prueba.
dotnet tool run playwright install chromium

Ejecutar pruebas de humo (Scripts de Postman):
newman run "Pruebas de humo - Hotel Osiris.json"
